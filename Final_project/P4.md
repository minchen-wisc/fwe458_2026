## Part 1 — Background

### HLS L30 Band Reference

The **Landsat-based** HLS collection (`HLSL30`) uses Landsat band numbering:

| Band | Wavelength | Used for |
|------|-----------|---------|
| B2 — Blue | 490 nm | RGB composites, BSI |
| B3 — Green | 560 nm | NDWI, MNDWI, RGB |
| B4 — Red | 665 nm | NDVI, EVI, SAVI, RGB |
| B5 — NIR | 865 nm | NDVI, EVI, SAVI, NDWI |
| B6 — SWIR-1 | 1610 nm | MNDWI, NDBI, BSI |
| B7 — SWIR-2 | 2190 nm | BSI, soil/mineral |
| Fmask | — | Cloud / shadow / snow quality mask |

### Three Classifiers

| Classifier | Complexity | sklearn class |
|-----------|-----------|--------------|
| **Gaussian Naive Bayes** | Simple — baseline | `GaussianNB()` |
| **Decision Tree** | Medium | `DecisionTreeClassifier(max_depth=10)` |
| **Random Forest** | Complex — ensemble | `RandomForestClassifier(n_estimators=200)` |

### Step 4: Compute Spectral Indices
