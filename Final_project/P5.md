## Overview

NASA's EMIT satellite is a visible-to-shortwave infrared (VSWIR) imaging spectrometer on the International Space Station, providing 285 contiguous spectral bands from 381–2493 nm at 60 m resolution. In this project, you will work with real EMIT surface reflectance data, compute hyperspectral vegetation indices, and build a machine learning model — either Partial Least Squares Regression (PLSR) or Random Forest (RF) — to estimate plant biochemical traits from spectra. Using your chosen model, you will run two modeling pipelines: one based on full reflectance spectra and one based on spectral indices. Comparing results across the two pipelines lets you evaluate whether hand-crafted spectral indices capture as much trait information as the full continuous spectrum. You will then produce regional trait maps and compare trait distributions across Plant Functional Types (PFTs).

This project is research-grade in difficulty and scope. Students completing it will gain experience with a workflow directly relevant to published literature in remote sensing of vegetation.

### Partial Least Squares Regression (PLSR)

PLSR is a multivariate regression method widely used in chemometrics and remote sensing. It projects both the predictor matrix **X** (e.g., reflectance spectra with hundreds of highly correlated bands) and the response variable **y** (e.g., leaf chlorophyll content) into a shared low-dimensional latent space, then performs regression on these latent components. Unlike ordinary least squares, PLSR handles multicollinearity and high-dimensional data gracefully — it can work even when the number of predictors far exceeds the number of observations. You select the number of latent components *n*, which controls model complexity; too few components underfit, while too many overfit. Cross-validation is typically used to choose the optimal *n*. In vegetation remote sensing, PLSR is the most common method for estimating biochemical traits (chlorophyll, nitrogen, water content, etc.) from full-spectrum reflectance data, because the full spectral shape — not just a few bands — carries information about leaf and canopy chemistry.

### Random Forest (RF)

Random Forest is an ensemble learning method that builds many independent decision trees, each trained on a bootstrap sample of the data with a random subset of features considered at each split. The final prediction is the average of all individual tree predictions (for regression). This double randomization reduces overfitting and makes RF robust to noisy predictors and nonlinear relationships. Key hyperparameters include the number of trees (`n_estimators`), the maximum depth of each tree, and the number of features considered per split (`max_features`). RF also provides a built-in measure of feature importance — features that reduce prediction error the most across all trees are ranked highest. In this project, RF offers an interpretable, nonparametric alternative to PLSR for estimating traits.

### Step 4: Hyperspectral Indices Computation

Compute the following spectral indices (or others of your choice). EMIT provides continuous spectra, so use the band nearest each target wavelength.

| Index | Formula | Wavelengths | Sensitive to |
|-------|---------|-------------|-------------|
| NDRE | (R750 − R705) / (R750 + R705) | 705, 750 nm | Red-edge chlorophyll |
| MTCI | (R754 − R709) / (R709 − R681) | 681, 709, 754 nm | Canopy chlorophyll content |
| REP | λ at max dR/dλ in 690–740 nm | 690–740 nm | Chlorophyll, stress |
| PRI | (R531 − R570) / (R531 + R570) | 531, 570 nm | Carotenoid:chlorophyll ratio |
| WBI | R900 / R970 | 900, 970 nm | Leaf/canopy water content |
| CAI | 0.5×(R2000+R2200) − R2100 | 2000, 2100, 2200 nm | Cellulose absorption |
| NDVI | (R800 − R670) / (R800 + R670) | 670, 800 nm | Green biomass / LAI |

> **Hint:** Write a helper function that finds the nearest EMIT band to a target wavelength:

```python
def get_band(wavelengths, target_nm):
    """Return the index of the EMIT band closest to target_nm."""
    return np.argmin(np.abs(wavelengths - target_nm))

# Example: compute NDVI
```

## Part 4 — Trait Estimation

### Step 7: Training Dataset Assembly

Build two modeling pipelines (using either PLSR or RF):

1. **Reflectance-based:** Full EMIT spectra (good bands only) as predictors → trait as target
2. **Index-based:** Hyperspectral indices as predictors → trait as target

Sample training points from your EMIT scene, extract reflectance and index values at each point, and match them with trait values from the trait database.

> **Hint:** Use 10-fold cross-validation (`sklearn.model_selection.KFold`) to evaluate models. For PLSR, use `sklearn.cross_decomposition.PLSRegression`. For Random Forest regression, use `sklearn.ensemble.RandomForestRegressor`.

```python
from sklearn.cross_decomposition import PLSRegression
from sklearn.ensemble import RandomForestRegressor
from sklearn.model_selection import cross_val_predict, KFold
from sklearn.metrics import mean_squared_error, r2_score

kf = KFold(n_splits=10, shuffle=True, random_state=42)

# Model 1: PLSR
pls = PLSRegression(n_components= int)
y_pred_pls = cross_val_predict(pls, X, y_trait, cv=kf)
rmse_pls = np.sqrt(mean_squared_error(y_trait, y_pred_pls))
r2_pls = r2_score(y_trait, y_pred_pls)
print(f"PLSR: R² = {r2_pls:.3f}, RMSE = {rmse_pls:.3f}")

# Model 2: RF
rf = RandomForestRegressor(n_estimators=200, random_state=42)
y_pred_rf = cross_val_predict(rf, X, y_trait, cv=kf)
rmse_rf = np.sqrt(mean_squared_error(y_trait, y_pred_rf))
r2_rf = r2_score(y_trait, y_pred_rf)
print(f"RF: R² = {r2_rf:.3f}, RMSE = {rmse_rf:.3f}")
```
