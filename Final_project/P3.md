## Part 1 — Background

### What is GCC?

The Green Chromatic Coordinate normalizes the green channel by total brightness:

> GCC = G_DN / (R_DN + G_DN + B_DN)

GCC ranges from 0 to 1 and is insensitive to illumination changes. Use `gcc_90` (90th percentile) rather than `gcc_mean` — it is more robust to cloud/shadow contamination.

### Phenological Metrics — Threshold Method

For each site-year, define:

- **winter_min:** 10th percentile GCC from January–February
- **peak_gcc:** maximum GCC from April–October
- **amplitude:** peak_gcc − winter_min
- **threshold_value:** winter_min + 0.25 × amplitude
- **SOS (Start of Season):** first spring day when smoothed GCC ≥ threshold
- **EOS (End of Season):** last autumn day when smoothed GCC ≥ threshold
- **GSL (Growing Season Length):** EOS − SOS in days

> `https://phenocam.nau.edu/data/archive/{site}/ROI/{roi_id}_3day.csv`

where `roi_id` follows the pattern `{site}_{PFT}_{NUM}` (e.g., `harvard_DB_1000`).

```python
import requests, io, pandas as pd, time, numpy as np

def download_gcc(site_name, roi_id, product='3day'):
    """
    Download 3-day GCC time series for a PhenoCam site.
    """
    url = (f"https://phenocam.nau.edu/data/archive/"
           f"{site_name}/ROI/{roi_id}_{product}.csv")
    resp = requests.get(url, timeout=30)

    if resp.status_code != 200:
        print(f"  [!] HTTP {resp.status_code} for {site_name}")
        return None

    # PhenoCam CSV has comment lines starting with '#' -- skip them
    df = pd.read_csv(io.StringIO(resp.text), comment='#')
    df['date'] = pd.to_datetime(df['date'])
    df = df.set_index('date').sort_index()
    print(f"  {site_name}: {len(df)} rows, {df.index.year.min()}-{df.index.year.max()}")
    return df

# Site name -> ROI ID mapping (PFT_NUM format)
```

### Step 2: Download DAYMET Climate Data

DAYMET provides 1 km daily climate data via a single-pixel API. The CSV response has 6 metadata lines to skip.

```python
def download_daymet(lat, lon, start_year=2015, end_year=2022,
                    variables='tmax,tmin,prcp,srad,vp'):
    """
    Download DAYMET single-pixel time series.
    DAYMET variables: tmax (°C), tmin (°C), prcp (mm/day), srad (W/m²), vp (Pa)
    """
    url = "https://daymet.ornl.gov/single-pixel/api/data"
    params = {
        'lat': lat, 'lon': lon,
        'vars': variables,
        'years': ','.join(str(y) for y in range(start_year, end_year + 1)),
        'format': 'csv'
    }
    resp = requests.get(url, params=params, timeout=60)
    df = pd.read_csv(io.StringIO(resp.text), skiprows=6)
    # Convert year + day-of-year to datetime
    df['date'] = pd.to_datetime(
        df['year'].astype(str) + df['yday'].astype(str).str.zfill(3),
        format='%Y%j'
    )
    df = df.set_index('date').sort_index()
    df['tmean'] = (df['tmax (deg c)'] + df['tmin (deg c)']) / 2
    return df
```

## Part 4 — Environmental Driver Analysis

### Step 5: Compute Climate Indices

For each site and year, compute from DAYMET:

- **Tmean_MA:** Mean temperature for March–April (spring forcing)
- **Tmean_DJF:** Mean temperature for Dec (prev year) + Jan–Feb (winter chilling)
- **GDD_spring:** Cumulative Growing Degree Days Jan 1 – May 31, base = 5 °C
- **Precip_MAM:** Total precipitation for March–May
