# Project 1 — Scientific Data Visualization

**Difficulty: 1 / 5 · Data: Provided datasets (CSV + GeoTIFF)**

## Overview

Build a portfolio of five publication-quality figures using Python, following the visual standards of leading scientific journals. You will be given a curated set of ready-to-use environmental datasets so you can focus entirely on the craft of data visualization.

## General Figure Standards

All five figures must follow these standards:

- Readable axis labels and tick labels
- Informative title or panel labels where appropriate
- Clean layout with no overlapping text
- Consistent font sizes
- Color choices appropriate to the data type
- Publication-quality resolution (300–600 dpi)
- Legends only when needed
- Units included where applicable

## Task 1 — Line Plot: Climate Time Series with Uncertainty Shading (20 pts)

**Dataset:** `GLB.Ts+dSST.csv`

Create a polished time-series figure showing long-term global temperature anomaly. Your figure must include:

- A line plot of annual or monthly temperature change over time
- Uncertainty shading or confidence interval band
- A running mean / moving average
- Correctly formatted time axis
- Horizontal reference lines at **0 °C** and **1.5 °C**
- Axis labels with units and a clear legend

## Task 2 — CONUS Map with Cartopy (20 pts)

**Dataset:** `2022_CONUS_land_cover_500m.tif`

Visualize the land cover raster on a projected map of the contiguous United States. Your figure must include:

- A correct map projection using **cartopy**
- Coastlines and state/national borders
- Raster visualization with an appropriate colormap
- A colorbar or discrete legend

Use `rasterio` to read the GeoTIFF and `cartopy` for the projected map. Pseudocode:

```
# 1. Open the GeoTIFF with rasterio
#    - read band 1 as a 2D array
#    - extract the CRS, bounds (left, bottom, right, top), and affine transform

# 2. Compute the geographic extent from the bounds
#    extent = [left, right, bottom, top]

# 3. Create a cartopy figure with a projected axes
#    - choose a projection suitable for CONUS (e.g., AlbersEqualArea or LambertConformal)
#    - set the map extent to cover CONUS
#    - add cartographic features: coastlines, state borders

# 4. Display the raster with ax.imshow()
#    - pass the extent and the source CRS as the transform argument
#    - choose an appropriate colormap (categorical for land cover)

# 5. Add a colorbar or discrete legend, title, and save at 300 dpi
```

> **Key libraries:** `rasterio` for reading GeoTIFFs, `cartopy.crs` and `cartopy.feature` for map projections and features, `matplotlib.colors.ListedColormap` for categorical colormaps.
>
> **Note:** If the raster CRS is not EPSG:4326 (lat/lon), you may need to reproject it first with `rasterio.warp.reproject` or adjust the `transform` argument in `ax.imshow()` accordingly.

## Task 3 — Scatter Plots (20 pts)

**Dataset:** `Chlorophyll_estimates.csv`

Create a scatter plot evaluating the relationship between observed and model-predicted chlorophyll content. Your figure must include:

- A scatter plot of predicted vs. observed values
- A **1:1 reference line** and a **fitted regression line**
- Annotated **R²**, **RMSE**, and **NRMSE** or **MAE**
- Points colored by density, model category, or another meaningful encoding

You may show a single model or compare 2–3 models in a multi-panel layout.

---

## Task 4 — Violin/Box Plot with Multi-Panel Layout (20 pts)

**Dataset:** `Leaf_Mass_per_area_measurements.csv`

Create a multi-panel figure combining a distribution plot with a second figure type. Your figure must include:

- A violin or box plot of **LMA by plant functional type**, with jittered raw data points overlaid
- Groups ordered by median LMA
- A second panel using a different figure type (e.g., spectral line plot, bar chart of group means)
- Panel labels **(a)** and **(b)** with consistent colors across panels

---

## Task 5 — Correlation Heatmap (20 pts)

**Dataset:** `Reflectance_data_by_two_sensors.csv`

Show the correlation structure among reflectance bands from two sensors. Your figure must include:

- A **lower-triangle** correlation matrix heatmap
- A diverging colormap centered at 0
- Annotated correlation values
- Readable variable names on both axes
